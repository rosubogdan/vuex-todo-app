# vuex-todo-app

## Project setup
```
yarn install
```

### Start json-server dummy db with hot-reload
```
yarn db
```
### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
